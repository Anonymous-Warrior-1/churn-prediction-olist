# Churn Prediction (Olist)

This repository contains a churn prediction case study notebook.

## Data
CSV files are not included. Download the Olist dataset from Kaggle and place it in a `data/` folder locally, then run the notebook.
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## How to run
Open `Churn Prediction.ipynb` and run cells top-to-bottom.
